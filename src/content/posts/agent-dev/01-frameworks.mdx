# 主流 AI Agent 开发框架分类

## 一、无代码 / 低代码应用生成

| 框架                  | 定位                   | 特点                               |
| --------------------- | ---------------------- | ---------------------------------- |
| **秒哒**        | 一句话生成应用         | 0 代码，自然语言 → 需求 → 应用   |
| **Dify**        | **开源 LLM 应用平台** | 可视化编排、完整开源方案、私有部署 |
| **Coze / 扣子** | **字节 Bot 平台**     | 插件生态丰富、快速搭建             |

## 二、桌面级 / 通用助理 Agent

| 框架                          | 定位             | 特点                       |
| ----------------------------- | ---------------- | -------------------------- |
| **搭子**                | 桌面级 AI 智能体 | 看屏幕、操作软件、处理文件 |
| **QoderWork**           | 桌面级 Agent     | 需求驱动，直接交付结果     |
| **Claude Computer Use** | 电脑操作 Agent   | Anthropic 官方，操作 GUI   |
| **Browser Use**         | 浏览器自动化     | 开源，专注 Web 操作        |

## 三、通用 Agent 开发框架（开发者向）

| 框架                    | 定位                | 特点                           |
| ----------------------- | ------------------- | ------------------------------ |
| **LangChain**     | **老牌通用框架**   | 生态最全，适合复杂推理链       |
| **LangGraph**     | **状态机式编排**   | 图结构、支持循环与长期状态     |
| **LlamaIndex**    | RAG + Agent         | 知识库、数据检索场景最强       |
| **Haystack**      | 搜索增强 Agent      | 企业搜索、文档问答             |
| **PydanticAI**    | 类型安全 Agent      | Python 类型约束，工程化友好    |
| **Mastra**        | 现代全栈框架        | Workflow + Memory + 部署可观测 |
| **Atomic Agents** | 模块化架构          | 强调可组合、可测试             |
| **DSPy**          | 声明式 Prompt/Agent | 推理过程可优化，Stanford 出品  |

## 四、厂商官方 SDK

| 框架                         | 出品方    | 特点                         |
| ---------------------------- | --------- | ---------------------------- |
| **Claude Agent SDK**   | Anthropic | 工具、MCP、子 Agent          |
| **OpenAI Agents SDK**  | OpenAI    | 轻量、Handoff、多 Agent 编排 |
| **Semantic Kernel**    | 微软      | 企业级、插件、多模型协同     |
| **Google ADK**         | Google    | 配合 Gemini                  |
| **AWS Bedrock Agents** | AWS       | 托管式、企业集成             |

## 五、多 Agent 协作框架

| 框架                         | 特点                                |
| ---------------------------- | ----------------------------------- |
| **AutoGen**            | 微软，动态对话式协作，研究界常用    |
| **CrewAI**             | 清晰角色体系驱动流程，易上手        |
| **MetaGPT**            | 模拟软件公司（产品/架构/开发/测试） |
| **Camel-AI**           | 角色扮演机制模拟团队                |
| **Swarms**             | 大规模 Agent 集群调度               |
| **Agno（原 Phidata）** | Agent + Knowledge + Tools 组合      |

## 六、自主任务执行 Agent

| 框架                                | 特点                                        |
| ----------------------------------- | ------------------------------------------- |
| **AutoGPT**                   | 现象级开源，Plan → Execute → Reflect 循环 |
| **BabyAGI**                   | 轻量目标驱动，教学价值高                    |
| **OpenHands（原 OpenDevin）** | 软件工程 Agent                              |

## 七、代码 / 工程类 Agent

| 框架                        | 特点                         |
| --------------------------- | ---------------------------- |
| **Claude Code**       | Anthropic 官方 CLI，终端编码 |
| **MonkeyCode**        | 云端开发环境，任务式 AI 编码 |
| **Aider**             | 终端 pair programming        |
| **SWE-agent**         | 学术界基准                   |
| **Cursor / Windsurf** | IDE 集成型                   |

## 八、工作流自动化 / 集成

| 框架                        | 特点                           |
| --------------------------- | ------------------------------ |
| **n8n**               | 集成面广、可自托管、系统对接强 |
| **Zapier AI Actions** | SaaS 集成王                    |
| **Make (Integromat)** | 可视化工作流                   |

## 九、多模态 / 内容生成 Agent

| 框架                    | 特点                                       |
| ----------------------- | ------------------------------------------ |
| **剪映-小云雀**   | 字节 Seedance 2.0 视频 + Seedream 5.0 图像 |
| **Runway / Pika** | 视频生成 Agent 化                          |

## 十、底层协议 / 基础设施

| 框架                                   | 特点                          |
| -------------------------------------- | ----------------------------- |
| **MCP (Model Context Protocol)** | Anthropic 推的工具/上下文标准 |
| **LiteLLM**                      | 统一多模型 API                |
| **Guardrails / Instructor**      | 输出结构化与安全约束          |

## 选型速查

| 你的需求              | 推荐                                                 |
| --------------------- | ---------------------------------------------------- |
| 一句话搞出应用        | 秒哒 / Dify / Coze                                   |
| 让 Agent 操作电脑     | 搭子 / QoderWork / Claude Computer Use               |
| 自动化打通内部系统    | n8n                                                  |
| 复杂推理链、深度定制  | LangChain / LangGraph                                |
| 知识库问答            | LlamaIndex / Haystack                                |
| 多角色协作            | CrewAI（易用）/ AutoGen（灵活）/ MetaGPT（软件流程） |
| 企业级 .NET/Java 环境 | Semantic Kernel                                      |
| 类型安全、工程化      | PydanticAI / DSPy                                    |
| Claude 生态开发       | Claude Agent SDK + MCP                               |
| AI 写代码             | Claude Code / MonkeyCode / Aider                     |
